# Custom Animated TabBar (React Native)

Ce projet est une implémentation personnalisée d'une barre de navigation inférieure (Bottom TabBar) avec animations, utilisant **React Native**, **React Navigation** et **React Native Reanimated**.  
Il propose un design moderne avec un highlight animé et des icônes dynamiques grâce à `@expo/vector-icons`.

---

## ✨ Fonctionnalités

- Barre de navigation personnalisée avec design arrondi et ombre portée
- Icônes dynamiques (Feather Icons) selon la route
- Animation fluide du highlight violet sous l’onglet actif
- Effet de zoom sur l’icône active avec `react-native-reanimated`
- Texte des onglets avec changement de couleur et animation d’opacité

---

## 📦 Packages installés

Voici les principales dépendances utilisées :

- **react-native** : framework mobile
- **expo** : environnement de développement simplifié
- **@react-navigation/native** : navigation principale
- **@react-navigation/bottom-tabs** : gestion des onglets
- **@expo/vector-icons** : pack d’icônes (Feather)
- **react-native-reanimated** : animations avancées
- **react-native-gesture-handler** : gestion des gestes (nécessaire pour React Navigation)
